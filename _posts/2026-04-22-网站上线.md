今天将个人主页从纯 HTML 迁移到了 Jekyll，搭建在 GitHub Pages 上。

## 为什么选择 Jekyll

1. **GitHub 原生支持**：推代码就自动构建，零 CI/CD 配置
2. **静态网站**：安全、快速、免费托管
3. **Markdown 支持**：写作流程更自然
4. **可扩展**：主题、插件、自定义布局

## 技术栈

- Jekyll 4.x
- GitHub Pages
- 自定义暗色主题
- Liquid 模板引擎
